## 🧑🏻‍💻 Installation

Run terminal first and check if installed Node.js by

```
$ node -v
```

Clone the project to local

```
$ git clone https://github.com/LJBL22/alpha-shop.git
```

Go to the directory

```
$ cd alpha-shop
```

Install npm

```
$ npm install
```

Install Sass

```
$ npm i sass
```

Start the project

```
$ npm start
```

You can now view alpha-shop in the browser at

```
$ localhost:3000
```

You can leave the project by

```
$ ctrl + C
```
